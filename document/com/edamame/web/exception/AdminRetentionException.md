# AdminRetentionException（例外仕様）

(このファイルは `document/web/exception/AdminRetentionException.md` から移動されました)

## 概要
`AdminRetentionException` は、最後の有効な `admin` アカウントを削除・無効化・ロール削除しようとした場合にスローされるカスタム例外です。サーバ側の最終防御機構として実装されています。
