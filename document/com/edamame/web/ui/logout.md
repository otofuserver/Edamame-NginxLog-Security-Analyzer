# Logout UI

作成日: 2025-12-29

概要:
- サイドバーのログアウトボタン（`#logout-btn`）に関するクライアント側の仕様。

仕様:
- ボタン押下で `/logout` へ POST（または GET でセッション破棄）を行う。
- 成功時は `/login?logout=success` へリダイレクト。
- クライアント側は `Logout` モジュールを通じてイベントハンドラを初期化する（`window.Logout.init()`）。

セキュリティ:
- CSRF 対策がある場合、ログアウトリクエストは CSRF トークンを付与する。

テスト手順:
1. ログイン状態で logout ボタンを押し、`/login?logout=success` に遷移することを確認する。
2. セッションが破棄され、認証が必要なページにアクセスすると `/login` にリダイレクトされることを確認する。
