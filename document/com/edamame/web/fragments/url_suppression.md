# url_suppression.html

対象: `src/main/resources/fragments/url_suppression.html`

## 概要
- URL抑止管理ビューの断片HTML。検索・サーバーフィルタ・一覧テーブル・新規作成ボタン・ページャを配置し、`url_suppression.js` で動的動作を提供する。

## 主な機能
- キーワード検索とサーバー別フィルタ（operator以上）。
- 抑止ルール一覧テーブル（サーバー/正規表現/有効/最終アクセス/破棄回数/更新日時）。
- ページャ表示（1ページ最大20件）。
- 新規作成モーダルのトリガー（adminのみ表示）。
- 行クリックでミニメニューを表示（`mini_menu.js`）。

## 挙動
- DOM読込後に `url_suppression.js` がサーバー一覧・検索実行・ページャ描画を行う。
- サーバーセレクト左にラベル「サーバー」を表示し、フィルタのサーバー選択は `すべて` を含���。
- 列は改行禁止・横スクロール許可・正規表現列を可変幅とし、ヘッダー/行の改行を抑止。

## 細かい指定された仕様
- ページャは `#url-suppression-pager` に設置し、`« ‹ page › »` を表示。
- モーダル（別途HTML内定義）はサーバー選択・正規表現・説明・有効チェックボックスを含む。
- 行が無効状態の場合は `inactive` クラスでグレー表示。

## その他
- ミニメニューの外観・位置計算は `mini_menu.js` に委譲し、スタイルは `style.css` で URL脅威度ビューと統一。
