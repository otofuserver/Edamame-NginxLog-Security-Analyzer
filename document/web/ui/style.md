# style.css（スタイル仕様）

ファイル: `src/main/resources/static/style.css`

## 概要
アプリケーションの共通スタイル定義を格納します。モーダル、テーブル、ボタン、フォーム要素のスタイリング、およびアクセシビリティに配慮したフォントサイズやコントラストを定義します。

## チェックポイント
- モーダルの `.modal` クラスおよび `.modal-backdrop` のスタイルが整っていること（表示・非表示切替時の視覚的崩れがない）
- テーブルの行ホバー・選択時のスタイルが視認性高く設定されていること
- 色は WCAG のコントラスト基準を満たすこと（重要なテキストやボタン）

## 推奨クラス
- `.btn`, `.btn-primary`, `.role-pill`, `.modal-content`, `.modal-backdrop` など

## テストケース
- モーダル開閉でレイアウト崩れが起きないこと
- テーブルの長いテキストが折り返される／省略される挙動の確認
