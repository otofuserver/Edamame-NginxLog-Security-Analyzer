# sidebar_mini_menu / サイドバーミニメニュー

対象: `src/main/resources/static/sidebar_mini_menu.js`（並びに関連 HTML 要素）

## 概要
- サイドバーのユーザーエリアに表示される小型メニュー（プロフィール、パスワード変更等）に関する UI 振る舞いとアクセシビリティ要件。

## 主な機能
- ミニメニューのトグル表示／非表示
- メニュー項目のクリックハンドリング（プロフィール、パスワード変更等）
- アクセシビリティ属性の更新（`aria-haspopup`, `aria-expanded`）

## 挙動
- `#sidebar-user-button` のクリックでミニメニューをトグル表示する。
- メニューの外側をクリックするとメニューを閉じる。
- メニュー項目は対応するモジュールへ遷移やモーダルをトリガーする（例：プロフィール → `ProfileModal.open()`）。

## 細かい指定された仕様
- `#sidebar-mini-menu` はルート要素として `#sidebar-user` 内に配置する。
- 表示状態は `aria-expanded` 属性で管理し、表示時は `true`、非表示時は `false` を設定する。
- キーボード操作（Enter/Escape）で開閉できること。

## その他
- テスト手順：サインイン時にユーザーアイコンをクリックして表示、外側クリックで閉じる、プロフィールクリックでプロファイルモーダルが開く等を確認。
