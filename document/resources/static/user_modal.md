# user_modal.js

対象: `src/main/resources/static/user_modal.js`

## 概要
- ユーザー編集モーダルのフロントロジックを提供する JavaScript ファイル。管理画面のユーザー編集フォームの開閉、フォーム送信、再送ボタン表示制御などを行う。

## 主な機能
- ユーザー情報の取得と編集フォームの初期化
- activation_tokens の状態により再送ボタンを表示/非表示にするロジックを実装
- フォーム送信時に `/api/users/{username}` へ PUT リクエストで更新を送信

## 挙動
- ユーザー詳細を取得すると `hasExpiredUnusedActivationToken` を参照し、編集モーダル下部に「アクティベーションメールを再送する」ボタンを表示する。
