# password_modal.js

対象: `src/main/resources/static/password_modal.js`

## 概要
- パスワード変更モーダルのフロントロジックを提供する。入力の検証、マスク解除、API 呼び出しを担当する。

## 主な機能
- 新しいパスワードと確認入力の検証
- マスク解除チェックボックスで表示切替
- バリデーションルール: 最低8文字、英字1文字以上、数字1文字以上、許可記号のうち1文字以上（許可記号に @ を追加）
- API 呼び出し: `/api/me/password` に POST で送信

## 挙動
- 入力が一致しない、もしくはポリシーに合致しない場合はエラーメッセージを表示して送信を中止する。
- 送信成功時はアラートで成功を通知しモーダルを閉じる。

## 細かい指定された仕様
- 許可記号セットは `!@#$%&*()-_@`（プロジェクト整合のため `@` を含める）。
- 許可外の文字が含まれる場合はエラーとする。
