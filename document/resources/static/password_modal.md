# password_modal / パスワードモーダル

対象: `src/main/resources/static/password_modal.js` と関連 HTML

## 概要
- ユーザーのパスワード変更および管理者によるリセット時のフロントエンドでの振る舞いを記述する。

## 主な機能
- 自己用パスワード変更フォームの表示と検証
- 管理者によるパスワードリセットのワンタイム表示とコピー支援
- マスク解除（`#password-unmask`）機能

## 挙動
- ユーザーが新しいパスワードを入力し、確認入力と一致することをチェックして API に送信する。
- 管理者リセット時はサーバ側で生成された一度きりのパスワードをフロントで表示し、ユーザーにコピーを促す。

## 細かい指定された仕様
- 最低 8 文字、英字/数字/記号を含むことをクライアント側検証で必須化。
- 許可記号は `!@#$%&*()-_`（プロジェクト方針により将来変更可能）。
- フロント側で生成したパスワードは一切保存しない（ログにも出力しない）。

## その他
- テスト手順：自己変更と管理者リセットの両フローを検証し、生成パスワードの表示が一度のみであることを確認。
