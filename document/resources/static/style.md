# style.css（スタイル仕様）

対象: `src/main/resources/static/style.css`

## 概要
- アプリケーションの共通スタイル定義（モーダル、テーブル、ボタン、フォーム、アクセシビリティ考慮）を記述する。

## 主な機能
- モーダルおよびバックドロップのスタイル管理
- テーブル表示（ホバー・選択・折り返し）のスタイル
- ボタン（.btn, .btn-primary）およびロール表示（.role-pill）のスタイル

## 挙動
- モーダル表示/非表示切替時に視覚崩れが発生しないこと
- 長いテキストは折り返しまたは省略（ellipsis）で適切に表示されること
- 色は WCAG コントラスト基準を満たしていること

## 細かい指定された仕様
- 主要 UI ボタンは十分なコントラスト（推奨比率 4.5:1 以上）を確保する。
- フォントサイズと行間はアクセシビリティ基準に従う（例: base font 16px 以上を推奨）。
- `.modal .modal-content` は max-width を設定し、レスポンシブ対応を行う。
- ミニメニューの外観は `mini_menu.css` に分離し、`@import './mini_menu.css'` で読み込む（style.css 側には持たない）。

## その他
